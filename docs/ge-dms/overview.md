# GE DMS

## What It Is
GE DMS (Dealer Management System) is the external GE system used to manage orders, inbound shipments, returns, and inventory reporting. The Warehouse app syncs data from specific GE DMS exports.

## How We Use It
- **ASIS**: load lists + model details exports
- **FG/STA**: ERP inventory export
- **Orders**: order download export
- **Inbound/Check‑In**: downloads page reports
