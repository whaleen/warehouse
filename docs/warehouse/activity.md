# Activity Log

## What It Is
The Activity Log shows a real‑time audit trail of key events.

## Common Entries
- ASIS sync events
- Load updates (friendly name, notes, color, prep flags, pickup date)
- Sanity check requests and completions

## How to Use It
1. Open **Activity** from the sidebar.
2. Scroll through recent events.
3. Use **Load more** to page older entries.

## Audience Notes

### For Developers
- Activity log shows sync and load updates.

### For Operators
- Use Activity to audit recent changes.

### For Agent
- Use for “what changed” questions.
