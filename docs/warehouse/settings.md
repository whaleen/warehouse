# Settings

## What Settings Controls
Settings manages locations, companies, team users, and personal preferences.

## Sections
- **Location Settings**: active location, SSO credentials
- **Company Profile**: company name and slug
- **Team**: manage users and roles
- **User Settings**: username, password, avatar
- **Displays**: manage floor display configurations

## Common Tasks
- Set active **Location**
- Update **SSO username/password** for GE sync
- Change **theme** and **UI handedness**
- Update **user profile** details

## Audience Notes

### For Developers
- Settings controls SSO creds, locations, users, displays.

### For Operators
- Use Settings to update location or user info.

### For Agent
- Use Settings for “where do I configure X”.
