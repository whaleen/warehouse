# Glossary

## Core Terms
- **ASIS**: GE ASIS inventory source for damaged/returned units.
- **FG**: Finished Goods inventory.
- **Local Stock (STA)**: Local warehouse stock inventory.
- **Sub‑Inventory**: Load/bucket identifier (used as load name on the floor).
- **CSO**: Customer order number from GE.
- **Serial**: Unique serial number on the unit.
- **Fog of War**: Scan mode that validates against inventory and updates GPS.
- **Ad‑hoc Scan**: Scan mode that accepts any barcode and logs a GPS marker.
- **Sanity Check**: Manual verification of load contents.

## Audience Notes

### For Developers
- Keep terms consistent with GE DMS exports.

### For Operators
- Use this to decode acronyms.

### For Agent
